# 谁是卧底 - Zoo Agent 游戏

## 游戏规则

1. **角色分配**：4人游戏，3平民+1卧底
2. **词语**：平民看到同一个词，卧底看到相似的词
3. **流程**：
   - 每人轮流描述（不能说词语本身）
   - 投票选出最可疑的人
   - 揭晓身份

## Agent 配置

| Agent | 角色 | 特点 |
|-------|------|------|
| 雪球 | 雪纳瑞 | 稳重、务实 |
| 六六 | 蓝鹦鹉 | 活泼、机灵 |
| 小黄 | 黄鹦鹉 | 温和、细心 |
| OpenAI | AI助手 | 逻辑强、分析型 |

## 实现方案

通过 agent 消息系统实现：
1. 游戏主持人（系统）分配词语
2. 轮流发言
3. 投票
4. 判定胜负
